# SoftwareProjectBackend

Frontend: https://github.com/yassienE4/SoftwareProjectFrontend

run project: npm run dev (also do "npm install" if its your first time)

save db changes: npx prisma migrate dev --name (put a name)

launch db editor: npx prisma studio --config ./prisma.config.ts

add this to copilot to load prisma docs: https://github.com/apps/prisma-for-github-copilot
